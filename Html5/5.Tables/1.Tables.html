Table

Table will display in row after row format

First tag will be "table"tag which is the opening of the table
* Inside table there is "tr" tag,which means you are creating a row from here
* Inside "tr"tag there will be a "td" tag==>which will create table cell.
* A "th" tag is like "td" but mostly "th" is used for headings.It also helps for search engines and for screen readers
* "th scope="col/row"> attribute will tell either the heading is for row or for coilumn
